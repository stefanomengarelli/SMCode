SMFront - C# Rapid Application Development Web Application Front End Code Library
---------------------------------------------------------------------------------

The goal of the SMFront project, currently under development, is to create a synthesis of the libraries designed over more than 25 years as support for web front end rapid application development.

SMCode project nuget URL: https://www.nuget.org/packages/SMCode
SMCode project GitHub URL: https://github.com/stefanomengarelli/SMCode
SMCodeJS project GitHub URL: https://github.com/stefanomengarelli/SMCodeJS
