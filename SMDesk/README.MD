SMDesk - C# Rapid Application Development Windows Form Desktop User Interface Code Library
------------------------------------------------------------------------------------------

The goal of the SMDesk project, currently under development, is to create a synthesis of the libraries designed over more than twenty years as support for desktop windows form rapid application development.

SMCode project nuget URL: https://www.nuget.org/packages/SMCode
SMCode project GitHub URL: https://github.com/stefanomengarelli/SMCode
